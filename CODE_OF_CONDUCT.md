# Code of Conduct

Elf3D project discussion and collaboration must remain professional, technical,
and respectful.

## Expected Behavior

- Focus criticism on code, evidence, requirements, and outcomes.
- Ask for clarification without assuming bad intent.
- Give specific, actionable feedback.
- Respect privacy, confidentiality, licensing, and ownership constraints.
- Use the minimum sensitive information necessary to investigate a problem.

## Unacceptable Behavior

- Harassment, threats, discrimination, personal attacks, or sustained
  disruption.
- Publishing private contact information, credentials, customer data,
  proprietary models, or license-ambiguous assets.
- Pressuring a person to disclose confidential material.
- Misrepresenting validation, authorship, license, or project status.

## Enforcement

Repository owners may remove material or restrict participation when behavior
violates this policy. Reports should be made privately through an existing
trusted channel and should include only the information needed to evaluate the
conduct.
