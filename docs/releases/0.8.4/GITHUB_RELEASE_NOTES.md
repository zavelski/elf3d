# Elf3D 0.8.4

Elf3D 0.8.4 refreshes the project's public presentation with a visual product
overview, an explicit architecture map, and clearer paths to the viewer,
documentation, and embeddable libraries.

## Download

- `elf3d-viewer-0.8.4-windows-x64.zip`
- `SHA256SUMS.txt`

The viewer package is a ready-to-run Windows application. GitHub also provides
source archives for the `v0.8.4` tag.

## Highlights

- Reworked the GitHub README around the practical viewer and library workflows,
  with quick links, product highlights, download and build instructions,
  current scope, and a concise repository map.
- Added a Mermaid composition diagram showing the `elf3d_viewer`,
  `elf3d_imgui`, `elf3d`, and `elf3d_model` products together with the nine
  grouped CMake module targets and their dependency direction.
- Added a real Elf3D Viewer screenshot featuring the Madame Walker Theatre
  model, credited to IUPUI University Library under CC BY 4.0.
- Added original Elf3D social-preview artwork and a dedicated README visual
  attribution record.
- Expanded the top-level third-party notice table to cover the licensed model
  depicted in the README screenshot.

## Compatibility

This release does not change the supported engine, model, ImGui integration,
viewer, or public API behavior. It updates project presentation, documentation,
and release metadata.

## Requirements

- Windows x64.
- OpenGL 4.1 core-profile graphics driver.
- Microsoft Visual C++ Redistributable for Visual Studio 2022.
