# Elf3D 0.7.6

Elf3D 0.7.6 publishes a clean public source tree, a Windows x64 viewer package,
and updated user documentation for scene navigation, selection, measurement,
clipping, building, testing, rendering, and C++ integration.

## Download

- `elf3d-viewer-0.7.6-windows-x64.zip`
- `SHA256SUMS.txt`

The viewer package is a ready-to-run Windows application. GitHub also provides
source archives for the `v0.7.6` tag.

## Highlights

- Improved rendering reliability for valid scenes that use very small nested
  model scales.
- Added a practical viewer guide and a complete viewer reference.
- Added public build, test, C++ API, glTF compatibility, and rendering
  documentation.
- Included the guide, viewer reference, model compatibility reference, and
  third-party notices in the Windows viewer package.

## Requirements

- Windows x64.
- OpenGL 4.1 core-profile graphics driver.
- Microsoft Visual C++ Redistributable for Visual Studio 2022.
