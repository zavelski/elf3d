# Support

Use GitHub Issues for reproducible product problems and documentation errors.

Include:

- the Elf3D release or source revision;
- Windows version, GPU, and graphics-driver version;
- the operation being performed;
- exact error text or relevant diagnostics;
- a small model you are permitted to share, when the problem is model-specific.

Do not attach confidential, customer-owned, personal, credential-bearing, or
license-restricted data. Security reports must follow `SECURITY.md`.
